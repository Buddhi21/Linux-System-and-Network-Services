# Linux System & Network Services – Assignment Project

### Systems & Network Programming – IE2012 (2025)

This repository contains a complete Linux administration and network services project, demonstrating practical experience in:

- Virtual machine setup and Linux installation
- File system navigation and user management
- DHCP, DNS, and NTP service configuration
- Shell scripting and cron automation
- SSH secure remote access
- Firewall management with iptables and ACLs
- Apache web server configuration
- Postfix email server setup
- GDB executable analysis, debugging, and file system monitoring

All tasks are performed on an Ubuntu and Kali virtual machine.
